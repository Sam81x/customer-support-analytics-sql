# customer-support-analytics-sql
## Overview
This project analyzes customer support operations using SQL to identify performance trends, resolution efficiency, and customer satisfaction drivers.

## Business Questions
- Which agents resolve tickets fastest?
- What categories have the longest resolution times?
- How does SLA compliance affect customer satisfaction?

## Tools
- PostgreSQL / MySQL
- Advanced SQL (CTEs, Window Functions, Joins)

## Key Skills Demonstrated
- Data modeling
- Complex joins
- Time-based analysis
- KPI calculation

- schema.sql
- queries.sql
